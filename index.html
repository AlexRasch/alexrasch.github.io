About Me

Hey there! I'm Alexander, I've recently completed a C# training program with a focus on web development through Academic Work. Programming has been a significant interest of mine for a long time, including languages like C#, PHP, and WordPress.

In addition to my technical background, I have experience in the construction industry and fire protection, where I've had the opportunity to work closely with clients and handle project management responsibilities.

I specialize in creating modern, scalable, and secure web applications using .NET and C#. I've utilized ASP.NET Core to build interactive web interfaces using technologies like Razor Pages and Blazor. Furthermore, I have expertise in HTML5, CSS, and JavaScript to craft responsive and user-friendly web pages.

Feel free to explore my portfolio to see examples of my work and projects. If you're interested in collaborating or have any questions, don't hesitate to reach out. Let's create something amazing together!
